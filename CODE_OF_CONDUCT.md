# Code of Conduct

NyayaAI follows the Contributor Covenant v2.1.

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

Applies to all project spaces — GitHub, Slack/Discord, review meetings, demos, and any public representation of the project.

## Enforcement

Report violations to the core team at `conduct@nyayai.app`. Reports are handled confidentially by a rotating two-person panel drawn from the four-person core team.

## Reporting algorithmic harm

NyayaAI exists to reduce algorithmic harm. If you believe a NyayaAI audit itself has caused harm to a person or group, report via the same channel with subject prefix `[HARM]`. We treat these with the highest priority.
